# Changelog

All notable changes to this project are documented in this file.

The project follows Semantic Versioning.

## 1.0.0 - 2026-07-14

### Added

- Automated Symfony 7.4 LTS dependency upgrades
- GitHub repository cloning and synchronization
- Dedicated dated security branches
- Composer dependency updates
- Composer validation
- Composer security auditing
- Abandoned-package reporting
- Symfony YAML validation
- Symfony dependency-injection container validation
- Twig template validation
- Conditional PHPUnit execution
- Automated Git commits and branch pushes
- Automated GitHub pull-request creation
- Safety protections against automatic merging and deployment
